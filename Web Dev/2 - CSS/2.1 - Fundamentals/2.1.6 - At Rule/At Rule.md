An **@rule** is a special CSS instruction that starts with `@` and tells the browser to **do something other than style a selector**.  
They control behavior, import other files, define conditions, or create reusable blocks.

### ðŸ§© **Common @rules**

|@rule|What it does|
|---|---|
|`@import`|Load another CSS file.|
|`@media`|Apply CSS only under certain conditions (screen size, etc.).|
|`@font-face`|Define custom fonts.|
|`@keyframes`|Define animations.|
|`@supports`|Apply CSS only if a feature is supported.|
|`@layer`|Control CSS cascade layers.|
|`@page`|Page-specific styles for printing.|

---

### ðŸ§  **Quick definition (one-liner)**

**@rules = CSS commands that control _how_ and _when_ styles apply, not styling elements directly.**
