- **What it is:** The setting you assign to the property.
- **Example:**
    - `#444444` is a dark gray color in hexadecimal format.
    - `21px` means font size is 21 pixels tall.
- **Other formats for values:**
    - Colors: `red`, `rgb(255,0,0)`, `hsl(0, 100%, 50%)`
    - Sizes: `em`, `rem`, `%`, `vh`, `vw`
