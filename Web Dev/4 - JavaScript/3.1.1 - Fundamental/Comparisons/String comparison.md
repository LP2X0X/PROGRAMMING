- Strings are compared letter-by-letter. 
- The above statement hold true when both string operands represent numbers.
	```js
	alert( 'Z' > 'A' ); // true
	alert( 'Glow' > 'Glee' ); // true
	alert( 'Bee' > 'Be' ); // true
	alert( "2" > "12" ) // true // Be careful!
	```
